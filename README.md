# Price Scout

## Overview
Price Scout is a price comparison tool that helps users find the best prices for products across various online retailers. The platform allows users to search for products, compare prices, and receive alerts when prices drop. 

## Features
- **Price Comparison:** Compare prices of products from different retailers.
- **Price Alerts:** Get notified about price drops for products you're interested in.
- **User Reviews:** Read and submit reviews for products to help inform purchasing decisions.
- **User-Friendly Interface:** Easy navigation and search functionalities to enhance user experience.

## Installation
To install Price Scout, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/czarinasablada02-collab/price-scout.git
   ```
2. Navigate into the project directory:
   ```bash
   cd price-scout
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To start the Price Scout application, run:
```bash
npm start
```
